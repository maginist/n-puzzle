## How to install

Create a virtual environment and activate it

On windows:

```bash
python -m venv venv
.\venv\Scripts\activate.bat
```

Install the pre-requirement:
```bash
pip install -U -r pre-requirements.txt
```

Execute `setup.py`:

* for dev:
```bash
pip install --editable .[dev]
```
* For user:
```bash
pip install .
```
